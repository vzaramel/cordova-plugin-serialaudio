
# vzaramel.cordova.dtmf

## Installation

    cordova plugin add https://github.com/vzaramel/cordova-plugin-dtmf.git

## Properties

- DTMF.startTone(succesCallBack, errorCallBack, int tone, int duration)

available tones are listed below:
 - DTMF.TONE_0
 - DTMF.TONE_1
 - DTMF.TONE_2
 - DTMF.TONE_3
 - DTMF.TONE_4
 - DTMF.TONE_5
 - DTMF.TONE_6
 - DTMF.TONE_7
 - DTMF.TONE_8
 - DTMF.TONE_9
 - DTMF.TONE_A
 - DTMF.TONE_B
 - DTMF.TONE_C
 - DTMF.TONE_D
 - DTMF.TONE_P
 - DTMF.TONE_S

### Supported Platforms

- Android
